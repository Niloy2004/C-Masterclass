# C Masterclass

A complete C programming course website built as a clean, single-page learning experience.

## Live Site

[https://niloy2004.github.io/C-Masterclass/](https://niloy2004.github.io/C-Masterclass/)

## About

C Masterclass is designed for beginner-to-advanced C programming practice. It includes structured chapters, lesson sections, examples, and a dashboard-style interface for learning C step by step.

## Highlights

- 12 chapters and 31 lessons
- Beginner-friendly explanations
- Code examples and structured sections
- Dark professional course interface
- Hosted publicly with GitHub Pages
