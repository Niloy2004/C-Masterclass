CHAPTER 01

C Language — Introduction & Basics

01  C Language Kya Hai? — History & Features ▾

C Language ka Janam

C ek general-purpose, procedural, compiled programming language hai. Isko Dennis Ritchie ne 1972 mein Bell Labs mein banaya — sirf UNIX operating system likhne ke liye!

Aaj bhi C duniya ki sabse important languages mein se ek hai. Linux kernel, MySQL, Python interpreter — sab C mein likhe hain.

C Ki Features — Kyun Seekhein?

  • Fast & Efficient — Machine ke bahut kareeb hai, bahut speed milti hai
  • Portable — Ek baar likho, kahin bhi chalao
  • Low-level Access — Memory directly manipulate kar sakte ho
  • Mother Language — C++, Java, Python sab issi se inspired hain
  • Structured — Functions aur modules se code organize hota hai
  • Rich Library — Bahut saare built-in functions available hain

Compiled vs Interpreted

C ek compiled language hai. Matlab aapka source code (.c file) pehle compiler (GCC/Clang) se machine code (.exe ya .out) mein convert hota hai, phir chalta hai.

💡 Yaad RakhoC ka development kuch is tarah hua: BCPL → B → C. C ko eventually C++ ne extend kiya.

08  Relational & Logical Operators ▾ 
Cint a = 5, b = 10;

printf("Max = %d\n", max);  // 10

09  Bitwise Operators — Bits Level Pe Kaam ▾

Bitwise — Binary Level Pe Operation

Cint a = 12; // binary: 1100

15  Recursion — Function Apne Aap Ko Call Karna ▾ 
C — Factorial/* n! = n × (n-1) × (n-2) × ... × 1 */
int factorial(int n) {
    if (n == 0 || n == 1) return 1;  // Base case
    return n * factorial(n - 1);      // Recursive case
}
/* factorial(5) = 5 * 4 * 3 * 2 * 1 = 120 */
C — Fibonacci/* 0, 1, 1, 2, 3, 5, 8, 13, ... */
int fib(int n) {
    if (n <= 1) return n;
    return fib(n-1) + fib(n-2);
}
C — Binary Search Recursiveint binarySearch(int arr[], int low, int high, int target) {
    if (low > high) return -1;  // Not found
    int mid = (low + high) / 2;
    if (arr[mid] == target) return mid;
    if (arr[mid] < target)  return binarySearch(arr, mid+1, high, target);
    return binarySearch(arr, low, mid-1, target);
}
⚠️ Base Case Zaroor Lagao!Recursion mein base case (stopping condition) nahi lagaya toh infinite loop — program crash kar dega (stack overflow)!

Pointers — C Ka Superpower

21  Pointers — Memory Addresses ▾

Pointer Kya Hai?

Pointer ek variable hai jo kisi doosre variable ka memory address store karta hai. Jaise ghar ka address — directly ghar pe nahi jaate, pehle address lete hain, phir jaate hain.

Cint x = 10;
int *ptr;        // pointer declare — * matlab "pointer to int"
ptr = &x;        // &x = x ka memory address

printf("x ka value: %d\n", x);     // 10
printf("x ka address: %p\n", &x);  // 0x7ffd... (memory address)
printf("ptr mein hai: %p\n", ptr); // same address
printf("*ptr = %d\n", *ptr);        // 10 — dereferencing

/* Value change karo pointer se */
*ptr = 50;
printf("x ab = %d\n", x);  // 50! (x change ho gaya)

/* NULL pointer — safe initialization */
int *nullPtr = NULL;  // kahi bhi point nahi karta
if (nullPtr != NULL) {
    // Tabhi use karo
}
⚠️ Dangling Pointer!Freed memory ko point karne wala pointer "dangling pointer" hai. Use karne se undefined behavior aata hai. Hamesha free ke baad ptr = NULL karo.

23  Pointers in Functions — Call by Reference ▾ 
C/* Call by Value — copy jaati hai, original nahi badle */
void addFive_val(int x) {
    x += 5;  // local copy change hoti hai
}

/* Call by Reference — address jaata hai, original badle */
void addFive_ref(int *x) {
    *x += 5;  // original variable change hota hai
}

void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int main() {
    int num = 10;
    addFive_val(num);     // num still 10
    addFive_ref(&num);   // num = 15 ✅
    
    int x = 5, y = 10;
    swap(&x, &y);
    printf("%d %d\n", x, y);  // 10 5
    return 0;
}

27  malloc, calloc, realloc, free ▾ 
C#include <stdio.h>
#include <stdlib.h>

/* malloc — uninitialized memory allocate karo */

/* sab elements 0 se initialized */

/* Use karo */
for(int i=0; i<n; i++) arr[i] = i * 10;

arr = (int*) realloc(arr, 10 * sizeof(int));

free(arr);

arr = NULL;  // dangling pointer se bachao
brr = NULL;
